# roblox-offsets
Welcome to my repo! on this repo you can find the offsets for roblox. If you have a auto updater service you can happily rely on this since I am a very fast updater!
Here is how to use the API on your project!

```python
import requests

version = "enterurversionhere!"

url = f"https://raw.githubusercontent.com/MoonArea3/roblox-offsets/refs/heads/main/{version}/offsets.json" # change it to hpp if you want to

offsets = requests.get(url).json()

print(offsets)
```
